ℹ️ How to Use Daily Task App

1. Plan Your Day (Home)

Start by adding rows for your daily tasks. Select a Category and a Task from the master lists. Set an Impact and Risk score (0-10) to calculate priority.

2. Manage Master Data

Go to the Categories page to create new Categories (e.g., 'Work', 'Personal') and add standard Tasks to them. This keeps your data organized and consistent.

3. Track Progress

As you work, update the % Done and Minutes. At the end of the day, click End Day.

  • Tasks < 100% are automatically kept for tomorrow.
  • Use Pin Mode to force tasks to stay (Recurring resets progress, Continue keeps it).

4. Analyze Information

Check the History to see past archives. Use the Analysis dashboard to see your performance metrics and completion rates over time.
